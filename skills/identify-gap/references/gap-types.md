**How this file is used.** `identify-gap` Phase 1 presents these types as a shared vocabulary — **presentation, not teaching** (decision-log 2026-06-04): the student arrives gap-literate from summary-writing, so the mentor names the types with their one-line definitions and moves on; definitions are unpacked only on request. Phase 2's mapping table uses these types as its third column. The framing follows the methods menu's philosophy: **current examples of what gaps look like, not a cage** — the list is non-exhaustive, the mentor is expected to recognize shapes beyond it in the student's actual materials, and a gap fitting no listed type is welcome.

## The types

1. **Open-question gap** — the question simply hasn't been asked in this literature.
2. **Contradiction gap** — studies disagree, and someone needs to adjudicate.
3. **Population / context gap** — an established finding, untested in a group, place, or setting.
4. **Temporal gap** — the findings predate something that matters (a platform, a policy, an era), or newer data now exists.
5. **Mechanism gap** — X→Y is established; *why* it happens is not.
6. **Moderator gap** — the effect is established on average; *for whom* it holds or breaks is not.
7. **Measurement gap** — the outcome or predictor *itself* is measured crudely or inconsistently; a better operationalization could change the conclusions. (If the measure is fine but the *analytic design* is too weak, that's item 8.)
8. **Methodological gap** — the literature's analytic designs limit what it can conclude; a different or stronger method could do better. Two of the Methods-related Gap's three forms live here: the **novelty form** (a method new to this literature) and the **re-application form** (an established method, aimed at the literature's open variables).
9. **Data-type gap** — the literature leans on one kind of data (say, self-report surveys); another kind (behavioral, text, geolocated) is untried.
10. **Replication / verification gap** — re-testing an established finding is itself a contribution: a new dataset, newer data from a tested dataset, or a different population are each legitimate by default — the original evidence does not need to be thin or contested for the re-test to matter. This is the Methods-related Gap's **replication form**.
11. **Intervention / evaluation gap** — the problem is well-described, but whether a policy, program, or intervention actually reduces it is untested (or tested only weakly). The motivating shape behind most program-evaluation work.
